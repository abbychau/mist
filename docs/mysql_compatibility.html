Common MySQL Features Not Supported

‚ùå Calculated Columns

General Database Limitations

  • In-memory only - No persistent storage
  • Single database - No multi-database support
  • No user management - No authentication or authorization
  • Limited concurrency - Thread-safe but no advanced locking
  • Memory usage - All data held in RAM
  • No stored procedures - Application logic only
  • No triggers - No event-driven logic
  • No views - Physical tables only

Migration from MySQL

‚úÖ Supported Migrations

  • Basic table structures with standard data types
  • Simple CRUD operations with basic WHERE conditions
  • Basic relationships with foreign keys
  • Simple reporting queries with aggregates and GROUP BY
  • Basic JOINs with equality conditions
  • UNION operations for combining results

‚ö†Ô∏è Manual Adjustments Needed

  • Replace function calls in SELECT with application logic
  • Convert calculated columns to application computations
  • Replace LIKE patterns with exact matches or application filtering
  • Replace subqueries with separate queries and application logic
  • Remove stored procedures, triggers, and views
  • Add explicit transactions where needed
