🚀 Complete MySQL Compatibility (Both Web & Native) SHARED ENGINE

✅ Core Features:

  • Complete DDL (CREATE/ALTER/DROP TABLE)
  • Full DML (INSERT/SELECT/UPDATE/DELETE)
  • All JOIN types (INNER, LEFT, RIGHT, CROSS)
  • Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
  • 35+ built-in functions (string, date, math)
  • Scalar subqueries in SELECT and WHERE clauses (correlated and non-correlated)
  • Advanced operators (LIKE, IN, BETWEEN, EXISTS with correlation)
  • Transactions with nested support and savepoints
  • Index management (CREATE/DROP INDEX)
  • UNION operations
  • Foreign key constraints with cascade
  • Auto-increment columns
  • All MySQL data types
  • Query recording and playback
Architecture Update: The web playground now uses the same SQL engine as the native MIST library, providing identical functionality through WebAssembly compilation.
